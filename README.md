# 🏠 Home Price Prediction

> **A Machine Learning web application that predicts residential property prices in Bengaluru based on user inputs such as location, total area, BHK, and bathrooms. The project combines data preprocessing, model training, and a Flask backend with a responsive web interface to deliver real-time price predictions.**

---

## 📖 Overview

Accurately estimating house prices is an important problem in the real estate industry. This project applies **Machine Learning** techniques to predict property prices in **Bengaluru** using historical housing data.

The application includes the complete machine learning pipeline, from data preprocessing and feature engineering to model training and deployment through an interactive web application.

---

## ✨ Features

▸ Predicts house prices based on user inputs.

▸ Trained using the Bengaluru House Price dataset.

▸ Performs data cleaning and feature engineering.

▸ Uses a trained Machine Learning model for prediction.

▸ Flask-based REST API for serving predictions.

▸ Interactive and responsive web interface.

▸ End-to-end Machine Learning deployment project.

---

## 📊 Dataset

The project uses the **Bengaluru House Price Dataset**, containing property details such as location, area, BHK configuration, bathrooms, and price.

### Dataset Files

| File | Description |
|------|-------------|
| `bengaluru_house_prices.csv` | Raw housing dataset |
| `columns.json` | Encoded feature columns |
| `banglore_home_prices_model.pickle` | Trained Machine Learning model |
